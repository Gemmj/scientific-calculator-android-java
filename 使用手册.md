# Android科学计算器使用手册

## 界面介绍

### 主界面布局
- **显示区域**
  - 上方：当前输入的数学表达式
  - 下方：计算结果（实时显示）
- **输入区域**
  - 数字键盘：0-9数字按钮
  - 运算符：+（加）、-（减）、×（乘）、÷（除）
  - 科学函数：sin、cos、tan、ln、log、√、^、!
  - 常量：π（圆周率）、e（自然常数）
  - 控制键：C（清除）、AC（全部清除）、⌫（退格）、=（等号）

### 菜单功能
- **历史记录**：查看和管理计算历史
- **清除历史**：删除所有历史记录

## 基础操作指南

### 数字输入
1. 点击数字按钮输入数值
2. 点击小数点(.)按钮输入小数
3. 支持多位数和小数的连续输入

### 基础运算
1. **加法运算**：输入数字 → 点击 + → 输入数字 → 点击 =
   - 例：计算 25 + 17
   - 操作：2 → 5 → + → 1 → 7 → =
   - 结果：42

2. **减法运算**：输入数字 → 点击 - → 输入数字 → 点击 =
   - 例：计算 50 - 23
   - 操作：5 → 0 → - → 2 → 3 → =
   - 结果：27

3. **乘法运算**：输入数字 → 点击 × → 输入数字 → 点击 =
   - 例：计算 8 × 6
   - 操作：8 → × → 6 → =
   - 结果：48

4. **除法运算**：输入数字 → 点击 ÷ → 输入数字 → 点击 =
   - 例：计算 72 ÷ 8
   - 操作：7 → 2 → ÷ → 8 → =
   - 结果：9

### 科学函数使用

#### 三角函数
- **正弦函数 (sin)**
  - 输入：sin(90)
  - 操作：sin → ( → 9 → 0 → ) → =
  - 结果：1（90度角的正弦值）

- **余弦函数 (cos)**
  - 输入：cos(60)
  - 操作：cos → ( → 6 → 0 → ) → =
  - 结果：0.5（60度角的余弦值）

- **正切函数 (tan)**
  - 输入：tan(45)
  - 操作：tan → ( → 4 → 5 → ) → =
  - 结果：1（45度角的正切值）

#### 对数函数
- **自然对数 (ln)**
  - 输入：ln(2.718)
  - 操作：ln → ( → 2 → . → 7 → 1 → 8 → ) → =
  - 结果：约等于1

- **常用对数 (log)**
  - 输入：log(100)
  - 操作：log → ( → 1 → 0 → 0 → ) → =
  - 结果：2

#### 其他函数
- **平方根 (√)**
  - 输入：√(16)
  - 操作：√ → ( → 1 → 6 → ) → =
  - 结果：4

- **幂运算 (^)**
  - 输入：2^3
  - 操作：2 → ^ → 3 → =
  - 结果：8

- **阶乘 (!)**
  - 输入：5!
  - 操作：5 → ! → =
  - 结果：120

### 数学常量
- **圆周率 (π)**
  - 直接点击π按钮
  - 结果：3.141592653589793

- **自然常数 (e)**
  - 直接点击e按钮
  - 结果：2.718281828459045

### 复合表达式计算
支持括号和运算优先级的复杂表达式：

1. **带括号的表达式**
   - 例：(2+3)×4
   - 操作：( → 2 → + → 3 → ) → × → 4 → =
   - 结果：20

2. **混合科学函数**
   - 例：sin(30) + cos(60)
   - 操作：sin → ( → 3 → 0 → ) → + → cos → ( → 6 → 0 → ) → =
   - 结果：1

## 高级功能

### 实时预览
- 在输入过程中，下方显示区域会实时显示当前表达式的计算结果
- 无需按等号即可预览结果

### 历史记录功能
1. **自动保存**：每次按等号计算的结果都会自动保存到历史记录
2. **查看历史**：
   - 点击右上角菜单按钮
   - 选择"历史记录"
   - 查看所有计算历史

3. **重用历史**：
   - 在历史记录界面点击任意记录
   - 该表达式会自动填入主界面
   - 可直接修改或重新计算

4. **清除历史**：
   - 在主界面点击菜单按钮
   - 选择"清除历史记录"
   - 确认删除所有历史

### 便民功能

#### 复制结果
- 长按计算结果区域
- 结果会自动复制到剪贴板
- 屏幕显示"结果已复制到剪贴板"提示

#### 清除操作
- **C键**：清除当前输入，恢复到上次计算结果
- **AC键**：清除所有内容，重新开始
- **退格键 (⌫)**：删除最后输入的一个字符

#### 错误处理
- 除零运算：显示"错误"
- 无效表达式：显示"错误"
- 数值溢出：显示"∞"（无穷大）

## 使用技巧

### 连续计算
1. 计算完成后，结果会自动作为下次计算的起始值
2. 可以直接输入运算符继续计算
3. 例：先算2+3=5，然后直接按×4，得到20

### 百分比计算
- 输入数字后按%键可转换为百分比
- 例：50% = 0.5

### 角度单位
- 所有三角函数使用角度制（度），非弧度制
- 例：sin(90°) = 1，而不是sin(π/2)

### 精度说明
- 计算结果保持高精度
- 显示时自动格式化，去除末尾无意义的零
- 整数结果不显示小数点

## 常见问题解答

### Q: 为什么cos(90)不等于0？
A: 由于浮点数计算精度问题，cos(90)的结果是一个非常接近0的极小数值，这在计算机数学运算中是正常现象。

### Q: 如何输入负数？
A: 可以用减号表示负数，例如：(-5) + 3 或者 0 - 5

### Q: 支持哪些运算优先级？
A: 遵循标准数学运算优先级：
1. 括号：()
2. 函数和幂运算：sin、cos、^等
3. 乘除：×、÷
4. 加减：+、-

### Q: 历史记录有数量限制吗？
A: 理论上没有限制，但建议定期清理以节省存储空间。

### Q: 可以输入多层括号吗？
A: 支持多层括号嵌套，例如：((2+3)×(4+1))×2
