# 安全政策

## 支持的版本

我们为安全漏洞发布补丁。哪些版本有资格接收此类补丁取决于 CVSS v3.0 评级：

| 版本    | 是否支持           |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## 报告漏洞

科学计算器团队和社区认真对待安全漏洞。我们感谢您负责任地披露发现的努力。

要报告安全问题，请使用 GitHub 安全公告中的 ["报告漏洞"](https://github.com/yourusername/scientific-calculator-android/security/advisories/new) 选项卡。

科学计算器团队将发送回复，说明处理您报告的后续步骤。在对您的报告进行初始回复后，安全团队将随时向您通报修复和完整公告的进展情况，并可能要求提供其他信息或指导。

请向维护第三方模块的人员或团队报告第三方模块中的安全漏洞。

## 安全更新流程

当收到安全漏洞报告时：

1. **确认**：我们将在 48 小时内确认收到您的漏洞报告。

2. **评估**：我们将在 7 天内确认漏洞并确定其影响。

3. **修复开发**：我们将开发和测试漏洞修复程序。

4. **发布**：我们将发布安全更新并公开披露漏洞详情。

5. **致谢**：我们将给予报告者适当的致谢（如果需要）。

## 安全最佳实践

使用此应用时：

- 保持应用更新到最新版本
- 仅从可信来源（GitHub 发布）下载应用
- 安装前审查应用权限
- 报告任何可疑行为

## 隐私注意事项

此计算器应用：
- 在您的设备上本地存储计算历史
- 不向外部服务器传输数据
- 不收集个人信息
- 适当使用标准 Android 权限

有关数据处理的更多信息，请参阅 README 中的隐私文档。
